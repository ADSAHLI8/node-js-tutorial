## Usage

This project uses MongoDb database for   backend.

### Install Dependencies
```bash
npm install nodemon
```
```bash
npm install express
```
```bash
npm install mongoose
```

### Run  App

The server will run on http://localhost:3000

```bash
npx nodemon app
```
